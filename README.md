<h1>Projet CRUD sur les personnages de manga</h1>

Technologie utilisé :
- Express
- Handlebars
- Mongoose
- Nodemon

Base de donnée MongoDB.

Pour executer le projet :

- Git clone 'projet'
- Ajouter la collection manga.JSON dans Mongo
- Ouvrir avec IDE
- executer la commande : nodemon server.js

Enjoy <3
